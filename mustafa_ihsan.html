What’s Included in the Starter Package

1) Cashier UI

  • Simple point-of-sale interface for recording sales.
  • Works with barcode scanning or manual item entry.
  • Supports cash and card transactions.
  • Receipt printing (basic support).

2) Cloud Sync

  • All sales data stored securely in Google Cloud.
  • Real-time updates so data is always safe and accessible.

3) Admin Access (Reports)

  • Basic sales reports (daily, weekly, monthly).
  • Export sales data for accounting (CSV/Excel).

4) Inventory Backend

  • Web-based admin panel to manage products and stock.
  • Add/edit/delete items with name, price, barcode, and quantity.
  • Automatic stock reduction after each sale.
  • Overview of remaining stock and simple stock reports.
